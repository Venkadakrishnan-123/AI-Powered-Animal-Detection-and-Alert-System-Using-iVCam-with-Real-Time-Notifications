# AI-Powered-Animal-Detection-and-Alert-System-Using-iVCam-with-Real-Time-Notifications
I developed an Intelligent Wildlife Detection and Alerting System to help reduce human–animal conflicts in areas near forests. The system uses AI to detect wild animals from a live camera feed and instantly sends alerts through a buzzer and SMS notification. It also stores detection details and images for future reference. This project combines AI, IoT, and real-time monitoring to improve safety for both people and wildlife.
